call,apply,bind使用 ：
call,apply,bind使用介绍及解析

call 的使用

指定内部 this 指向

  • 函数的 call 方法可以实现将函数内部 this 指向call 的第一的参数，然后调用该函数，并以 ‘,’ 为分割进行传参

进行函数的数据借用

  • 这里可以利用借用构造函数把 Person 中的数据初始化 Teacher 实例上

进行方法借用

  • 本来 obj 伪数组不能使用 forEach 和 push 方法，采用 call 将方法对象指向 obj,使之可以使用

apply 的使用

指定内部 this 指向

  • 函数的 apply 方法可以实现将函数内部 this 指向 apply 的第一个参数，然后调用该函数
  • apply 以数组进行传参，apply 会把数组中的元素一个一个的拿出来作为函数的参数，并且 apply 只处理一个参数和第二个传参参数，其它的不处理

进行方法借用

  • 本来数组不能使用 Math.max 方法，采用 apply 将方法对象指向 arr,使之可以使用

bind 的使用

指定内部 this 指向

  • 函数的 bind 方法可以实现将函数内部 bind 指向 bind 的第一个参数，函数不会被立即调用，会返回新的函数，新的函数内部 this 指向 bind 的第一个参数

bind 的传参方式

  • bind 可以传参数，但是不调用，得到 bind 之后的新函数在调用的时候也可以传参数，当你在调用通过 bind 得到的新函数的时候，实际上会把你在 bind 的时候传递的参数和你调用新函数传递的参数进行合并，然后作为函数的参数，bind 的参数在前，新函数的参数在后，若超过实际函数传参的数量，则只取前面的参数
