数组方法整理

js
  • 数组

数组方法整理 ：
关于数组方法

join()

  • 把数组中的所有元素转化为一个字符串，并以指定分隔符分割
  • Array.join(separtor);
    • 可选，指定字符作为分割符；如果省略，则使用逗号作为分割符
  • 返回一个字符串

  • 原数组不变

  • 通过join()方法可以实现重复字符串，只需传入字符串以及重复的次数，就能返回重复后的字符串，函数如下：

push()

  • 可以接收任意数量的参数，把它们逐个添加到数组末尾
  • Array.push(1,2,3);
  • 返回修改后数组的长度
  • push方法会直接添加新的元素到数组尾部，原数组会被修改

pop()

  • 用于删除数组的最后一个元素
  • Array.pop();
  • 返回被删除的最后一个元素，如果数组为空则返回 undefined
  • 原数组会被改变

shift()

  • 用于删除数组的第一个元素
  • Array.shift();
  • 返回删除的第一项的值，如果数组为空则返回 undefined
  • 原数组会被改变

unshift()

  • 可以接收任意数量的参数，把它们逐个添加到原数组开头
  • Array.unshift(1,2,3);
  • 返回修改后数组的长度
  • 原数组会被改变

reverse()

  • 反转数组项的顺序
  • Array.reverse();
  • 返回反转后的数组
  • 原数组会改变

字符串方法 split()

  • 把一个字符串分割成字符串数组
  • string.split(separator,howmany);
    • separator 必须传递，可以为字符串或正则表达式，从改参数指定的位置分割
    • howmany 可选传递，指定返回数组的最大长度，如果不设置，则整个字符串都会被分割
  • 返回一个字符串数组

  • 如果 把空字符串(“”)作为 separator 进行传递，那么目标字符串的没个字符之间都会被分割，作用正好与 array.join() 相反

  • 不适用遍历方法，将字符串”12345”反序输出

sort()

  • 按升序排列数组项——即最小的值位于最前面，最大的值排在最后面
  • Array.sort();
  • 返回排序完成的数组
  • 原数组被改变

  • 在排序时，sort()方法会调用每个数组项的 toString()转型方法，然后比较得到的字符串，以确定如何排序，但这种方式存严重问题，就是只比较第一位的大小

  • 为了解决这个问题，sort()方法可以接收一个比较函数作为参数，以便我们指定哪个值位于哪个值的前面。

  • 比较函数有两个参数，如果第一个参数小于第二个参数则返回一个负数，如果两个参数相等则返回 0，如果第一个参数大于第二个参数则返回一个正数。

concat()

  • 将参数添加到原数组中(类似push)
  • Array.concat();
  • 返回新的经过处理的数组
  • 原数组没有改变

  • 这个方法会先创建当前数组一个副本，然后将接收到的参数添加到这个副本的末尾，最后返回新构建的数组。在没有给 concat() 方法传递参数的情况下，它只是复制当前数组并返回副本。

  • 传入的不是数组，则直接把参数添加到数组后面，如果传入的是数组，则将数组中的各个项添加到数组中。

slice()

  • 返回从原数组中指定开始下标到结束下标之间的项组成的新数组
  • 可以接受一或两个参数，即要返回项的起始和结束位置
    • 在只有一个参数的情况下， slice()方法返回从该参数指定位置开始到当前数组末尾的所有项
    • 如果有两个参数，该方法返回起始和结束位置之间的项——但不包括结束位置的项

  • 原数组不会改变

  • 当参数为负数时，即为从后向前的位置

splice()

  • 一个顶四个，可以实现删除、插入和替换
  • 删除：可以删除任意数量的项，只需指定 2 个参数：要删除的第一项的位置和要删除的项数，不包含第二个参数的元素。
  • 插入：可以向指定位置插入任意数量的项，只需提供 3 个参数：起始位置、 0（要删除的项数）和要插入的项。
  • 替换：可以向指定位置插入任意数量的项，且同时删除任意数量的项，只需指定 3 个参数：起始位置、要删除的项数和要插入的任意数量的项。插入的项数不必与删除的项数相等。
  • splice() 方法始终都会返回一个数组，该数组中包含从原始数组中删除的项，如果没有删除任何项，则返回一个空数组。
  • 原数组会被改变

indexOf() 和 lastIndexOf()

indexOf()

  • 查找数组中的指定项
  • 接收两个参数：要查找的项（可以是字符串）和（可选的）表示查找起点位置的索引。从数组的开头（位置 0）开始向后查找。
  • 返回要查找的项在数组中的位置,在没找到的情况下返回 -1,相当于全等于检测
  • 原数组不会被改变

lastIndexOf()

  • 查找数组中的指定项
  • 接收两个参数：要查找的项（可以是字符串）和（可选的）表示查找起点位置的索引。其中，从数组的末尾开始向前查找
  • 返回要查找的项在数组中的位置,在没找到的情况下返回 -1,相当于全等于检测
  • 原数组不会被改变

forEach()

  • 对数组进行遍历
  • 参数是function类型，函数默认有传参，参数分别为：遍历的数组内容；第对应的数组索引，数组本身
  • 没有返回值

map()

  • 对数组中的每一项运行给定函数,函数参数即为数组元素
  • 返回每次函数调用的结果组成的数组
  • 原数组没有变化

filter()

  • 对数组中的每一项运行给定函数,函数参数即为数组元素
  • 返回满足过滤条件组成的数组
  • 原数组没有变化

every()

  • 对数组中的每一项运行给定函数,函数参数即为数组元素
  • 判断数组中每一项都是否满足条件，只有所有项都满足条件(每一项函数的返回都是true)，才会返回true
  • 原数组没有变化

some()

  • 对数组中的每一项运行给定函数,函数参数即为数组元素
  • 判断数组中是否存在满足条件的项，只要有一项满足条件(只要有一项函数的返回都是true)，就会返回true
  • 原数组没有变化

reduce() 和 reduceRight()

  • 这两个方法都会实现迭代数组的所有项，然后构建一个最终返回的值
  • reduce()方法从数组的第一项开始，逐个遍历到最后。而 reduceRight()则从数组的最后一项开始，向前遍历到第一项。
  • 接收两个参数：一个在每一项上调用的函数和（可选的）作为归并基础的初始值
  • 第一个参数函数有四个参数：前一个值、当前值、项的索引和数组对象
  • 这个函数的返回值会作为第一个参数自动传给下一项的函数
  • 第一次迭代发生在数组的第二项上
  • 原数组没有变化时
