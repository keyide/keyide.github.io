函数作用域 ：
关于函数作用域

JavaScript 作用域

  • JavaScript 中没有块（花括号）作用域
  • 对象和函数拥有私有作用域
  • try-catch拥有私有作用域
  • 一般服务端要根据异常捕获到的错误信息去记录错误日志，方便系统错误排查， Node.js 会大量的接触这个 try-catch

作用域链

  • 如果当前作用域内存在要访问的变量，则使用当前作用域的变量，否则到上一层作用域内寻找，但不会进入同级函数的私有作用域，直到全局作用域，如果找不到，则该变量为未声明

函数的作用域链以定义时所处的作用域为准

  • 函数的作用域链以定义时所处的作用域为准，而不是调用时的
