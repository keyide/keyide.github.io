关于函数内部 this 指向问题

function 中的 this

  • 具体指向谁取决于函数的调用方式。
  • 普通的函数调用内部 this 指向 window,严格模式下是 undefined
  • 构造函数调用，内部 this 指向函数实例,原型方法中 this 也是实例对象
  • 调用对象中的函数，this 指向该方法所属对象

事件处理函数中的this

  • 这里把 Person 作为 btn.click 的事件处理函数事件处理函数不是用户来调用的，而是由系统来调用事件处理函数内部的 this 指向 DOM 对象

定时器函数中的 this

  • 这里也是将 Person 作为定时器的处理函数当事件到达，系统会去帮你调用这个 Person，所以，这个定时器处理函数中的 this 指向的是 window

函数内this指向不受外部环境影响

  • 函数内 this 指向不受外部环境影响

数组遍历方法中的 this

  • 对于数组的一些遍历方法：
    例如 forEach、map、filter、some 等，内部的 this 都指向 window
    无论这个代码的执行处于什么环境都是 window,且不受外部影响

原型函数中的 this

  • 原型函数中的 this 指向函数原型

原型函数中的 this

  • 谁点了函数，this 就指向谁
